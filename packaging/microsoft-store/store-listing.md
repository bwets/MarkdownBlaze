## Description

MarkdownBlaze is a fast, no-fuss viewer for Markdown (`.md`) files. Point it at a document and it
renders a clean, readable page in a moment — then gets out of your way.

Built for reading docs, READMEs, notes, and wikis, it understands the Markdown you actually use:
GitHub-flavored tables and task lists, fenced code with syntax highlighting for ~190 languages,
Mermaid diagrams, callout/admonition blocks (both Docusaurus and MkDocs styles), and YAML front
matter. Local images are shown inline.

It's a real viewer, not just a renderer. A pinnable, resizable sidebar gives you a live outline of
the current document plus your reading history, and clicking a Markdown link opens that file right
inside the app — so a folder of linked notes behaves like a small wiki. Edit a file in your editor
and the view refreshes automatically.

MarkdownBlaze runs completely offline. There are no accounts, no ads, and no tracking — it doesn't
send your documents or any data anywhere. Everything it needs to render is built in.

### Highlights
- Open any `.md` / `.markdown` file (set it as your default Markdown handler if you like)
- Syntax-highlighted code blocks (~190 languages)
- Mermaid diagrams — flowcharts, sequence, class, timelines, and more
- Admonitions / callouts (tip, note, warning, danger, …)
- Live document outline, Back/Forward, and a persistent global history
- In-app navigation between linked Markdown files; folder links resolve to `index.md`
- Auto-refresh on file changes
- Light, Dark, and System themes
- Keyboard shortcuts (Back/Forward, Refresh, Print, toggle sidebar)
- Fully offline — private by design
